In the **Worklog**, I summarize the entire learning and practical process throughout the 12-week internship in the **First Cloud AI Journey** program as a **Cloud Infrastructure Engineer** in the **Smart Healthcare Appointment System** project. The content is structured with Weekly Goals, Implementation Tasks, and Achievements to strictly follow the requirements of the internship report template.

The entire process is arranged from familiarizing with **AWS**, learning basic cloud architecture knowledge and labs, participating in workshops and office activities, to designing Cloud Native architectures, deploying, configuring, and managing the **AWS Cloud** infrastructure for the **Smart Healthcare Appointment System**.The internship progress is divided by week as follows:

* **Week 1:** Explore AWS and Core Services
* **Week 2:** Explore Compute and Storage with EC2, Auto Scaling, Lightsail, S3, EBS, EFS, Storage Gateway, and Snow Family
* **Week 3:** Explore CloudFront, Advanced Networking, ALB, RDS, DynamoDB, and ElastiCache
* **Week 4:** Explore Lambda, API Gateway, SQS, SNS, CloudWatch, and Serverless Lab
* **Week 5:** Explore Secrets Manager, KMS, CloudFormation, CodeBuild, CodePipeline, and CI/CD
* **Week 6:** Explore AWS Security and Containerization with Security Hub, GuardDuty, WAF, Cognito, ECR, ECS, and Fargate
* **Week 7:** Explore Route 53, HTTPS, Monitoring, Governance, Event-Driven Serverless, Backup, Cost Optimization, and the Well-Architected Framework
* **Week 8:** Explore AI/ML with Amazon Bedrock, SageMaker AI, Rekognition, Textract, and Comprehend; Improve Architecture Diagrams
* **Week 9:** Design 3-Tier Cloud Architecture and Deploy Networking, NAT Instances, and S3 for Smart Healthcare Appointment System
* **Week 10:** Deploy RDS PostgreSQL, EC2 Application Servers, ALB, SQS, DLQ, and S3 Event Notifications
* **Week 11:** Finalize CloudFront, CloudWatch, SNS, S3 Lifecycle, and Verify Overall Infrastructure Integration
* **Week 12:** Test, Security Review, Cost Optimization, Project Summary, and Finalize Internship Report
