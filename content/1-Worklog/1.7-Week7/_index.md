### Week 7 Objectives:

* Configure DNS and HTTPS for applications deployed on AWS.
* Improve monitoring, auditing, and governance capabilities using AWS operational services.
* Learn about event-driven serverless integration patterns.
* Learn about backup, disaster recovery, and cost optimization.
* Evaluate AWS architectures using the AWS Well-Architected Framework.

### Week 7 Achievements:

* Configured DNS and handled HTTPS certificate validation for applications on AWS.
* Used CloudTrail, AWS Config, and CloudWatch for auditing, compliance, and operational monitoring.
* Understood event-driven processing with EventBridge, Step Functions, Lambda, and SQS/SNS.
