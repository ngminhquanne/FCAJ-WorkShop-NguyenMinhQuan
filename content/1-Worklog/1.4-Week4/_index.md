### Week 4 Objectives:

* Learn about the serverless computing model with AWS Lambda.
* Build HTTP APIs using Amazon API Gateway.
* Learn about asynchronous integration mechanisms with Amazon SQS and Amazon SNS.
* Monitor serverless applications using Amazon CloudWatch.
* Build and test serverless APIs using API Gateway, Lambda, DynamoDB, and CloudWatch.

| Day | Tasks | Start Date | Completion Date | Reference Material |

### Week 4 Achievements:

* Created and invoked AWS Lambda Functions with appropriate runtimes and execution roles.
* Built HTTP APIs using Amazon API Gateway and integrated them with Lambda.
* Understood the differences between Amazon SQS message queues and Amazon SNS pub/sub notification mechanisms.
* Configured CloudWatch Logs, Metrics, and Alarms to monitor serverless resources.
* Completed the serverless Hands-on Lab using API Gateway, Lambda, DynamoDB, and CloudWatch.
* Practiced checking IAM permissions, verifying request flows, and cleaning up AWS resources.
